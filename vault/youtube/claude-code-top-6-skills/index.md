# 클로드 스킬 100개 써봤는데, 진짜는 이 6개입니다

> https://www.youtube.com/watch?v=5rbzj5IUA78

## 요약

100개 이상의 클로드(Claude) 스킬을 직접 사용해본 뒤, 설치 후에도 꾸준히 손이 가는 진짜 유용한 6개 스킬만 추려 소개하는 영상이다. 소개된 스킬은 Ponytail, Caveman, Taste Skill, Matt Pocock Skills, ECC, last30days(+Superpowers)이며, 각 스킬의 설치 방법과 실제 작동 방식을 직접 시연한다. 핵심 기준은 '화려하게 보이는 스킬'이 아니라 실제로 반복 사용하게 되는 실용성이다.

## 인사이트

- Ponytail은 '게으른 시니어 개발자' 컨셉으로, 코드 작성 전 필요성 체크리스트를 통과한 기능만 최소한으로 구현해 불필요한 코드 팽창을 방지한다.
- Caveman은 클로드의 출력 답변을 원시인 말투처럼 압축해 읽는 속도를 높이고 출력 토큰을 줄이지만, 사고(thinking) 토큰을 줄이는 것은 아니다.
- Taste Skill은 'AI 슬롭(AI slop)' 문제를 해결하는 안티슬롭 프레임워크로, AI에게 디자인 취향을 부여해 레이아웃·애니메이션 수준을 1~10단계로 조절할 수 있다.
- Matt Pocock Skills는 개발 프로세스를 통째로 가져가는 대형 프레임워크와 달리, 조합 가능하고 수정하기 쉬운 단위 스킬로 구성되어 있으며 핵심은 작업 전 요구사항을 명확히 하는 'grill-me' 스킬이다.
- 스킬 선택 기준은 설치 직후 한 번 써보는 것이 아니라, 이후에도 계속 반복 사용하게 되는지의 실용성이다.
- 여러 스킬이 공통적으로 토큰 절약을 목표로 하지만, 접근 방식은 다양하다: 코드 최소화(Ponytail), 출력 압축(Caveman), CLAUDE.md 파일 압축(Caveman compress) 등.
- Taste Skill은 단일 스킬이 아니라 소프트, 미니멀리스트 등 여러 스타일 서브스킬로 구성되어 있어 원하는 디자인 언어를 선택해 적용할 수 있다.
