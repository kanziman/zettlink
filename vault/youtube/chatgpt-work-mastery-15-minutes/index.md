# 15분만에 챗지피티 워크 마스터하기

> https://www.youtube.com/watch?v=biZsmM5kEcA

## 요약

이 영상은 ChatGPT Work(워크)를 활용해 신제품 런칭 전 과정을 자동화하는 방법을 15분 안에 실습으로 안내합니다. 캔들 브랜드의 겨울 신제품 출시를 예제로 삼아, 프로젝트 생성부터 플러그인 설치, 스킬 등록, 컨셉 문서 작성, 광고 이미지 생성, 랜딩 페이지 배포, 문의 메일 자동 답장까지 전 과정을 순서대로 따라합니다. 특히 브랜드 톤을 '스킬'로 한 번 등록하면 이후 모든 결과물에 자동으로 반영되는 점이 핵심 기능으로 소개됩니다. Google Drive·Gmail 플러그인 연동을 통해 외부 문서를 참조하고 결과물을 로컬 폴더에 저장하는 워크플로우를 구성할 수 있습니다. 영상 시청 후 업무 시간이 두 배 이상 단축될 것이라고 강조합니다.

## 인사이트

- ChatGPT Work는 단순 채팅이 아니라 '결과물을 만들고 저장하는 작업 공간' 개념으로, 로컬 폴더를 프로젝트로 지정해 산출물을 관리한다.
- 플러그인(Google Drive, Gmail 등)을 설치하면 외부 서비스와 연동해 문서 참조 및 메일 자동 처리가 가능해진다.
- '스킬' 기능은 브랜드 가이드처럼 반복적으로 참조해야 하는 정보를 한 번 등록해 두면 이후 모든 작업에 자동 적용되는 영구 컨텍스트 역할을 한다.
- 컨셉 문서 → 광고 이미지 → 랜딩 페이지 → 문의 메일 초안까지 단일 툴 안에서 연속된 워크플로우로 처리할 수 있어 작업 전환 비용이 제거된다.
- 프로젝트 폴더명은 영문으로 지정해야 인식 오류를 방지할 수 있는 실용적 팁이 중요하다.
- 구글 드라이브의 참고 문서와 로컬 작업 폴더를 분리 운영하면 원본 자료와 산출물을 체계적으로 관리할 수 있다.
- 문의 메일 자동 답장 초안 기능을 통해 반복적인 고객 커뮤니케이션 업무를 매일 아침 자동화할 수 있다.
