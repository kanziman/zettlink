# 15분만에 챗지피티 워크 마스터하기

> https://www.youtube.com/watch?v=biZsmM5kEcA

## 요약

이 영상은 ChatGPT Work(워크)를 활용해 신제품 런칭 전 과정을 자동화하는 방법을 15분 안에 실습형으로 안내합니다. 캔들 브랜드의 겨울 신제품 출시를 예시로, 프로젝트 생성부터 플러그인 설치, 스킬 설정, 컨셉 문서 작성, 이미지 생성, 랜딩 페이지 배포, 문의 메일 자동 답장까지 전 과정을 다룹니다. ChatGPT Work 데스크톱 앱에서 Google Drive·Gmail 플러그인을 연결하고, 브랜드 가이드를 '스킬'로 저장하면 이후 모든 결과물에 브랜드 톤이 자동 적용됩니다. 영상에서 소개된 결과물은 모두 AI가 생성한 것으로, 작업 시간을 두 배 이상 단축할 수 있다고 강조합니다.

## 인사이트

- ChatGPT Work는 단순 대화(채팅)를 넘어 결과물을 저장하는 작업 공간 개념으로, 로컬 폴더를 프로젝트로 지정해 산출물을 체계적으로 관리할 수 있다.
- Google Drive·Gmail 플러그인을 연결하면 외부 문서를 ChatGPT Work 내에서 직접 불러와 활용할 수 있으며, 설치는 로그인 및 권한 허용만으로 완료된다.
- '스킬(Skill)' 기능은 브랜드 가이드처럼 매번 반복 설명하기 귀찮은 정보를 한 번 등록해 두면, 이후 모든 작업에 자동 적용되는 일종의 재사용 가능한 컨텍스트 프리셋이다.
- 컨셉 문서 → 이미지 생성 → 랜딩 페이지 배포 → 이메일 자동 답장으로 이어지는 마케팅 워크플로우 전체를 ChatGPT Work 하나로 연결할 수 있다.
- 프로젝트 폴더명은 영문으로 설정하는 것을 권장하며, 한글 폴더명 사용 시 인식 오류가 발생할 수 있다.
- 승인 요청 옵션이 활성화된 경우 플러그인 액션마다 수동 승인이 필요하므로, 자동화 수준에 맞게 설정을 조정해야 한다.
- ChatGPT Work는 모든 플랜에서 데스크톱 앱을 통해 사용 가능하며, 웹 버전에서는 상단 메뉴에서 '워크'를 선택해 접근할 수 있다.
