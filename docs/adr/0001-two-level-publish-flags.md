# 발행 플래그를 카드와 산출물 두 레벨로 분리

카드(`index.md`)와 각 산출물(`deep.md`, `til.md`, `guide.md`)은 발행 목적이 다르다. 카드 요약본은 지식 아카이브로서 공개하고, TIL은 학습 블로그로 공개하지만, 가이드는 본인 프로젝트 메모로 비공개 상태가 기본이다. 따라서 단일 플래그로 카드 전체를 제어하는 대신, 카드 레벨 `published` 플래그로 요약본을 공개하고, 산출물마다 개별 `published` 플래그를 두어 선택적으로 공개한다.

## Considered Options

- **카드 단위 발행**: 플래그 하나로 요약본+모든 산출물 일괄 공개. 단순하지만 가이드 같은 개인 메모도 강제 공개된다.
- **산출물 단위 발행**: 요약본 포함 모든 파일에 개별 플래그. 유연하지만 "카드를 공개한다"는 개념이 없어진다.
- **하이브리드 (채택)**: 카드 발행 = 요약본 공개. 산출물은 각자 별도 발행.
